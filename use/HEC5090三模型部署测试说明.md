## 5. 真机运行前检查

HEC-5090 盘点时 `enp11s0` 没有 carrier，`enp12s0` 为 UP。插好 G1 网线后必须重新确认，不能按接口名猜测：

```bash
ip -br link
ip -br address
export ROBOT_NET=enp11s0  # 改成实际连接 G1 且显示 LOWER_UP 的接口
```

以下真机命令会初始化 DDS 并可能发送 LowCmd。必须使用吊架、现场急停和遥控器，机器人周围无人。
